LOCAL TESTING
1. Unzip this package.
2. Open Terminal / Command Prompt inside the unzipped folder.
3. Run one of these commands:

   Mac:
   python3 -m http.server 8000

   Windows:
   py -m http.server 8000

4. Open this in your browser:
   http://localhost:8000

For most simple static hosting, upload the full folder contents exactly as-is.
The homepage file must stay named index.html.

DOMAIN
After hosting, connect AKAYASERENITY.COM in your hosting provider by updating DNS records.
